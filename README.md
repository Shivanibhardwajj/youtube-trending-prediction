# YouTube Trending Video Prediction

## Project Overview

This project predicts whether a YouTube video is likely to become trending by analysing video metadata and engagement metrics. The objective is to identify the factors that influence trending videos and build machine learning models capable of predicting trending status.

This project was completed as part of my MSc dissertation in Advanced Computer Science with Data Science at the University of Strathclyde.

---

## Objectives

- Analyse characteristics of trending and non-trending videos
- Perform Exploratory Data Analysis (EDA)
- Engineer meaningful features
- Train and compare multiple machine learning models
- Evaluate model performance using standard classification metrics

---

## Dataset

The project uses:

- YouTube Trending Dataset
- Additional metadata collected using the YouTube Data API
- Non-trending videos collected for comparison
